# Amore Liya 

## What's this about?
Track Baby Liya's Diet

## Installation

```bash
$ npm install
$ npm start

# Or for the dev mode
$ npm run dev
```

Open http://localhost:8080/

## Deploy to GCP
```bash
$ gcloud config set project amoreliya
$ gcloud app deploy
$ gcloud app browse
```

Quickstart for Node.js in the App Engine[https://cloud.google.com/nodejs/getting-started/hello-world]
